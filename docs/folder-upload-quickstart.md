# 文件夹上传功能 - 快速使用指南

## 🎯 功能亮点

现在你的云盘支持：
- 📁 **点击上传文件夹** - 保留完整文件夹结构
- 📁 **拖拽上传文件夹** - 直接拖拽整个文件夹
- 📁 **自动创建层级** - 系统自动创建所有子文件夹

## 🚀 快速开始

### 方式 1：点击上传文件夹

1. 在文件列表页面，点击 **"上传文件夹"** 按钮
2. 选择你想要上传的文件夹
3. 等待上传完成

![点击上传示意](示意图：用户点击"上传文件夹"按钮，选择整个项目文件夹)

### 方式 2：拖拽上传文件夹

1. 从电脑中拖拽一个文件夹
2. 拖拽到云盘页面任意位置
3. 看到彩虹边框提示后松开鼠标

![拖拽上传示意](示意图：用户拖拽文件夹到云盘页面)

## ⚠️ 注意事项

1. **浏览器兼容性**
   - Chrome、Edge、Firefox：完全支持
   - Safari：建议使用点击上传方式

2. **大文件夹上传**
   - 建议分批上传超大文件夹
   - 单个文件大小限制取决于服务器配置

3. **文件夹命名**
   - 如果同名文件夹已存在，会使用现有文件夹
   - 文件会直接上传到对应位置

## 🔧 技术细节

- 使用 HTML5 File API
- 支持相对路径解析
- 自动递归创建文件夹层级
- 异步处理确保顺序正确

## 📞 问题反馈

如果遇到问题，请检查：
1. 浏览器是否支持文件夹上传
2. 文件夹大小是否超限
3. 网络连接是否稳定
